## 5. 주요 발견 사항

- **선/선분 전체 QA 통과 0건**: 이식 완료(100%)됐으나 21개 전체가 실패 또는 건너뜀. 기능 작동 방식 전반의 점검이 필요한 수준.
- **함수 생성 시 화면 이탈 문제 반복**: 점 기반 함수 다수(삼차·사차·포물선·쌍곡선 등)에서 생성 위치에 따라 끝 조절점이 화면 밖으로 벗어나는 동일 패턴 버그 확인.
- **부속 점 미생성 패턴**: 타원(꼭짓점·초점), 쌍곡선(꼭짓점·초점) 등 생성 시 함께 생성되어야 할 부속 점이 나타나지 않는 문제 반복.
- **점근선 처리 미흡**: 지수·로그·탄젠트·유리함수에서 점근선이 독립 객체가 아닌 단일 객체로 묶이거나 무한 길이로 생성되어 조정 불가.
- **미이식 카테고리 다수**: 입체도형, 텍스트/이미지, 확률 탁자 순열, 일괄조정, 경로찾기 5개 카테고리 전체 미이식(총 39개).
- **파일/저장 위험**: 9개 중 7개 미이식으로 기본 저장·불러오기 UX 요건 미충족.

## 7. 개발자 권고사항

### 즉시 조치 권고

1. **수직선(직선→점 선택) 오작동 수정** — 수선 기능으로 잘못 작동 중. 의도한 동작 명확화 필요.
2. **직사각형·대각마름모 오작동 수정** — 두 기능 모두 평행사변형이 생성됨. 로직 혼용 의심.
3. **원의 평행접선 오류 수정** — 실행 시 화면 사라짐 발생.
4. **파일/저장 이식 계획 수립** — 불러오기, 내보내기(불러오기 연동 포함) 등 기본 저장 UX 없이는 오픈 불가.

### 중기 개선 제안

1. **함수 생성 위치 초기화 로직 공통 수정** — 삼차/사차/포물선/쌍곡선 등 다수에서 반복되는 화면 이탈 버그. 공통 초기 위치 계산 로직 검토 권장.
2. **타원·쌍곡선 부속 점 생성 누락 수정** — 꼭짓점·초점 등이 함께 생성되어야 하는 항목들 일괄 점검.
3. **점근선 처리 방식 개선** — 무한 길이 생성 및 독립 객체 미분리 문제. 지수·로그·탄젠트·유리함수 공통 대응 필요.
4. **캔버스/편집 기능 이식 우선순위 확정** — 다중 선택, 드래그 이동 등 기본 편집 기능 이식 일정 공유 요청.

### 확인 요청 사항

- [ ] 잘라내기(CUT)와 분할 기능이 동일한 기능인지 확인 요청
- [ ] 내보내기(json) 후 다시 불러오는 방법 — 현재 불러오기 기능 미이식 상태로 활용 불가한지 확인
- [ ] 객체 선택 순서 자유도 (선분 선택 시 선분에 대응하는 두 점으로도 선택 가능하게 할 계획인지)
- [ ] 선/선분 카테고리 전체 QA 실패에 대한 개발자 검토 결과 공유 요청
