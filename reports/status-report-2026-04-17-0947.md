## 7. 개발자 권고사항

### 즉시 조치 권고

1. **파일/저장 기능 이식 착수**: 저장·내보내기 기능 없이는 사용자 오픈 불가. 9개 항목 이식 일정 확인 요청.
2. **캔버스/편집 기능 이식 확대**: 현재 20% 수준으로 기본 편집 환경 미완성. 우선순위 높은 편집 기능(실행취소, 복사, 삭제 등) 이식 요청.

### 중기 개선 제안

1. **선/선분 실패 항목 일괄 검토**: 18개 실패 — 동일 원인인지 개발자 확인 후 일괄 수정 가능 여부 검토.
2. **점 기반 함수(41개) QA 착수 준비**: 이식률 66%(27/41). 이식 완료 27개에 대한 QA를 다음 기간 시작 권장.

### 확인 요청 사항

- [ ] 입체도형(8개) — 웹버전 이식 계획 있는지? 데스크톱 전용으로 확정되면 항목 분리 처리 예정.
- [ ] 일괄조정(14개), 경로찾기(2개) — 이식 로드맵 공유 요청. 현재 전량 미이식.
- [ ] 텍스트/이미지(5개), 확률 탁자 순열(10개) — 이식 예정 시점 확인 요청.
- [ ] 선/선분 전반적 실패 원인 — QA 데이터 공유 후 개발자 1차 확인 요청.
