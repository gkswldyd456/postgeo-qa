## 5. 주요 발견 사항

- **점 계열(Points)**은 전체 25개 중 100% 이식 확인 완료로 가장 진척이 좋음. QA도 집중 진행되어 실패(8건)·건너뜀(2건) 포함 전수 검증 완료.
- **선/선분(Lines)**도 100% 이식 확인. 단, 4개 partial 항목 모두 오류 또는 기능 차이 존재.
- **평면도형(Plane Figures)**은 이식률 73.5%이나 partial 9개 중 대부분에서 오동작(다른 도형 출력, 오류) 확인됨. 실질 사용 가능 항목은 이식률보다 낮을 수 있음.
- **캔버스/편집**은 이식률 20%로 저조. 객체 이동·다중 선택 등 기본 편집 UX가 대부분 미이식 상태.
- **파일/저장** 전체 미이식은 사용자 오픈 전 반드시 해결해야 할 블로커.
- **입체도형, 일괄조정, 경로찾기, 텍스트/이미지, 확률 탁자 순열** 이식률 0% — 우선순위 검토 필요.

## 7. 개발자 권고사항

### 즉시 조치 권고 (블로커)

1. **파일/저장(File & Save)** 기능 이식 — 저장·불러오기 없이는 실사용 불가
2. **캔버스 기본 편집** 이식 우선 — 객체 선택(다중), 이동 등 핵심 편집 UX

### 중기 개선 제안

1. **대각마름모·직사각형** 오출력 수정 (현재 평행사변형이 나옴)
2. **원호의 등분점** 오류 수정 및 n등분 기능 완성
3. **길이점선** (3점형·선분형·원호형) 실행 오류 수정
4. **포물선** — 준선 선택 방식 데스크톱과 동일하게 수정

### 확인 요청 사항

- [ ] 입체도형(3D Figures) 8개 — 웹 버전 지원 계획 여부

- [ ] 현재 기능 실행 시 안내 문구 없음 — 사용자 UX 개선 필요 (전체 코멘트 참고)
