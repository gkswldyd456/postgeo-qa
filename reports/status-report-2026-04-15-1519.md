## 7. 개발자 권고사항

### 즉시 확인 권고

1. **파일/저장 기능 이식 일정** — 저장 없이는 실사용 불가. 우선순위 및 일정 공유 요청.
2. **l-17 원의 평행접선** — 실행 시 화면 소실 + 오류. 다른 partial 오류 중 가장 심각한 동작 이상.
3. **pf-22 대각마름모 / pf-23 직사각형** — 완전히 다른 도형(평행사변형)이 표시되는 명백한 오류.

### 중기 개선 제안

1. **길이점선 3종 (pf-28/29/30)** — 모두 실행 안됨. 일괄 수정 가능성 검토.
2. **n등분 기능 2종 (p-16/32, l-06)** — 등분 수 입력 UI 미구현. 동일 패턴으로 일괄 수정 가능할 듯.
3. **객체 이동/선택 (ce-01/02)** — 다중 선택 및 이동 범위 확장 필요. 편집 UX 핵심 기능.

### 확인 요청 사항

- [ ] 입체도형(3D Figures) 8종 — 웹 버전 지원 계획 있는지?
- [ ] 화살표1/2, 회전축 화살표 (fp-32/33/34) — 실제 기능인지, 아니면 테스트 항목?
- [ ] 이차함수 (꼭짓점과 이차항계수, fp-35) — 신규 추가 기능인지?
- [ ] 점근선 기능 (fp-12 등) — 함수에 종속된 토글로 처리할 계획인지? (테스터 의견: 독립 기능보다 딸린 기능이 적합해 보임)
