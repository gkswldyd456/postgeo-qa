# PostGeo QA Dashboard

PostGeo 웹버전 기능 이식 현황 및 QA 상태를 추적하는 대시보드입니다.

## 접속 방법

```
https://gkswldyd456.github.io/postgeo-qa/
```

토큰 없이 누구나 열람 가능합니다. 상태 저장은 토큰 보유자만 가능합니다.


## 대시보드 기능

| 기능 | 설명 |
|------|------|
| 상태 뱃지 클릭 | 웹이식 / QA 상태 순환 변경 |
| 💾 저장 | GitHub에 변경사항 저장 (병합 저장 — 동시 작업 안전) |
| 🔄 변경 감지 배너 | 다른 편집자가 저장하면 20초 내 알림 |
| 필터 | 카테고리 · 웹이식 · 구현상태 · QA 상태별 필터링 |
| 컬럼 리사이즈 | 헤더 경계 드래그로 너비 조정 |
| 컬럼 가이드 ⓘ | 각 컬럼 헤더 클릭 시 체크 기준 팝오버 |
| 편집 모드 ✏ | 항목 이름 수정·삭제 / 카테고리 추가·수정·삭제 (삭제 시 항목 전체 삭제 경고) |
| 드래그앤드롭 (항목) | 편집 모드 ⠿ → 같은 카테고리 내 순서 변경 또는 다른 카테고리로 이동 (ID 자동 변경) |
| 드래그앤드롭 (카테고리) | 편집 모드 카테고리 헤더 ⠿ → 카테고리 순서 변경 |
| 인라인 삽입 | 편집 모드에서 행 사이 hover → ＋ 위치 추가 |
| 이슈/메모 | 셀 클릭으로 이슈 번호 + 메모 편집 |
| 전체 코멘트 탭 | 특정 기능과 무관한 전반적 의견 기록 |
| 코멘트 타입 관리 | 설정에서 타입 추가/삭제 |

## 편집 권한 설정

### 토큰 보유자 (직접 설정)
1. 대시보드 **⚙ 설정** → GitHub 사용자명 + Personal Access Token 입력
2. Fine-grained PAT 권장: `Contents: Read and write` 권한만 부여

### 비개발자 편집자 (공유 링크 방식)
GitHub 계정 없어도 아래 형식의 URL을 한 번 열면 자동으로 편집 권한 설정:

```
https://gkswldyd456.github.io/postgeo-qa/#ghowner=gkswldyd456&ghtoken=토큰값
```

최초 1회만 이 URL로 접속 → 이후엔 일반 URL로 접속해도 편집 가능.

## 동시 작업 안내

여러 명이 동시에 편집해도 자동으로 병합 저장됩니다.

- **서로 다른 항목** 편집 → 저장 시 자동 병합, 데이터 유실 없음
- **같은 항목** 편집 → `_modified` 타임스탬프 기준, 더 늦게 수정한 내용이 반영됨
- **카테고리 추가** → 양쪽 모두 추가한 카테고리 합집합으로 보존
- **카테고리 삭제** → 로컬에서 삭제한 카테고리는 병합 시 제거됨
- **카테고리·항목 순서** → 로컬 순서 우선, 원격에서만 추가된 항목은 원격 상대 위치 보존
- **변경 감지** → 다른 편집자가 저장하면 20초 내 상단 배너로 알림

> 같은 카테고리에서 동시에 항목 이동·삭제 작업은 피하는 것을 권장합니다.
